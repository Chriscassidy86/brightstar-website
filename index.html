Create a New Repository

Now let's create a repository for your website:

  1. Go to github.com and log in
  2. Click the "+" icon in the top right and select "New repository"
  3. Name it "brightstar-website" (or any name you prefer)
  4. Make sure it's set to "Public" (this is free)
  5. Optionally, add a description like "My BrightStar Solutions website"
  6. Click "Create repository"
Note: Don't initialize the repository with a README unless you're familiar with Git.
3

Upload Your HTML File

Once your repository is created, you need to upload your website files:

  1. In your repository, click "Add file" â†’ "Upload files"
  2. Drag and drop your index.html file or click to select it
  3. Add any other files if you have them (CSS, JS, images)
  4. Click "Commit changes"
Important: Make sure your main file is named index.html (all lowercase) as this is what Vercel will look for.
4

Sign Up for Vercel

Now let's get your website deployed:

  1. Go to vercel.com
  2. Click "Sign up" and choose "Continue with GitHub"
  3. Follow the authorization steps to connect your GitHub account
  4. Complete your profile information
5

Import Your Project

After signing up, you'll be taken to your Vercel dashboard:

  1. Click "Add New" â†’ "Project"
  2. You'll see a list of your GitHub repositories
  3. Find and select your "brightstar-website" repository
  4. Click "Import"
6

Deploy Your Website

Final step - deploy your website:

  1. Vercel will automatically detect your project settings
  2. Click "Deploy" and wait a few seconds
  3. Your website will be live at a URL like: brightstar-website.vercel.app
Success! Your BrightStar Solutions website is now live on the internet!

Troubleshooting Tips

  • Make sure your file is named index.html (all lowercase)
  • Check the Vercel deployment logs if you encounter any errors
  • Your site will automatically redeploy when you make changes to your GitHub repository
  • If you have CSS/JS in separate files, make sure the file paths are correct
